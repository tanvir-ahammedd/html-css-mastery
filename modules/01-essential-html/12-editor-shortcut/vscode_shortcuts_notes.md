# VS Code Keyboard Shortcuts – Notes

---

## UI & General Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+P` | Open Command Palette — search for any command or setting |
| `Ctrl+B` | Toggle sidebar (file explorer) |
| `Ctrl+J` or `Ctrl+~` | Toggle terminal |
| `Ctrl+F` | Search in current file |
| `Ctrl+Shift+F` | Search across all files |
| `Ctrl+S` | Save file |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+C` | Copy *(works without selecting — copies the whole line)* |
| `Ctrl+X` | Cut *(same as above, no selection needed)* |
| `Ctrl+V` | Paste |
| `Ctrl+Shift+N` | Open new editor window |
| `Ctrl+Shift+T` | Reopen last closed tab |
| `Ctrl+Tab` | Switch between open tabs |
| `Ctrl+W` | Close current file/tab |

---

## Code Editing Shortcuts

| Shortcut | Action |
|---|---|
| `Alt+↑ / Alt+↓` | Move current line up or down |
| `Shift+Alt+↑ / Shift+Alt+↓` | Copy (duplicate) line up or down |
| `Alt+← / Alt+→` | Move cursor by word (faster horizontal navigation) |
| `Home` | Go to beginning of line |
| `End` | Go to end of line |
| `Shift+Home` | Select from cursor to beginning of line |
| `Shift+End` | Select from cursor to end of line |
| `Shift+Arrow` | Select character by character |
| `Shift+Alt+Arrow` | Select word by word |
| `Ctrl+D` | Select next occurrence of current selection |
| `Ctrl+Shift+L` | Select all occurrences of current selection |
| `Ctrl+/` | Toggle comment on current line |
| `Ctrl+W` | Expand selection to enclosing tag/scope *(use repeatedly to expand outward)* |
| `Ctrl+Enter` | Insert new line below |
| `Ctrl+Shift+Enter` | Insert new line above |

---

## Tips

- **Command Palette is your best friend** — if you don't know how to do something, `Ctrl+Shift+P` and search for it.
- **Minimize mouse usage** — using keyboard shortcuts makes you significantly faster and more productive.
- **Copy/Cut without selecting** — placing your cursor anywhere on a line and pressing `Ctrl+C` or `Ctrl+X` copies/cuts the entire line.
- **`Ctrl+W` expand selection** — especially useful in HTML when you have deeply nested tags and want to quickly select an entire element.
